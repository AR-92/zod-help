# Installation Guide

This guide will walk you through installing the Zod Harness on your system.

## Prerequisites

Before installing, ensure you have:

- **Node.js** 18+ or **Bun** 1.3+
- **Git** for version control
- **Internet connection** for downloading packages

## Installation Methods

### Using Bun (Recommended)

```bash
# Install Bun if you haven't already
curl -fsSL https://bun.sh/install | bash

# Install Zod Harness
bun add @zod/harness
```

### Using npm

```bash
npm install @zod/harness
```

### Using yarn

```bash
yarn add @zod/harness
```

## Verify Installation

After installation, verify it worked:

```bash
npx zod --version
```

You should see the version number printed.

## Initial Setup

Run the setup wizard:

```bash
npx zod init
```

This will:

1. Create a configuration file
2. Set up your project structure
3. Guide you through initial configuration

## Troubleshooting

If you encounter issues:

- Ensure your Node.js version is 18 or higher
- Check that you have write permissions in your project directory
- Verify your internet connection is stable
- Review the error message and check our [troubleshooting guide](/troubleshooting/installation)
