Welcome to Zod Help Center

Welcome to the official Zod Help Center! Whether you’re new to AI agents or an experienced developer, you’ll find everything you need here to get started with the Zod platform.

What is Zod?

Zod is an open-source AI agent platform that allows you to create, manage, and deploy intelligent agents that can automate tasks, answer questions, and interact with various systems.

Quick Start Guide

  1. Create an account - Sign up at zodagent.com
  2. Install the harness - Follow our installation guide to set up your environment
  3. Create your first agent - Use the Zod Studio interface or CLI to build agents
  4. Explore features - Check out our feature guides to learn what’s possible

Platform Overview

Zod consists of several key components:

  • Zod Harness - The core framework for building and running agents
  • Zod Studio - Visual interface for agent management and monitoring
  • Zod Plugins - Extend agent capabilities with custom plugins
  • Zod Client - SDK for integrating agents into your applications
