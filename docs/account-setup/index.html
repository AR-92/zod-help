Account Setup

How to create and configure your Zod account

Account Setup

Learn how to create your Zod account and configure your profile settings.

Creating an Account

  1. Visit zodagent.com
  2. Click Sign Up in the top right corner
  3. Choose your registration method:
    • Email and password
    • GitHub OAuth
    • Google OAuth
  4. Verify your email address
  5. Complete your profile setup

Account Types

Free Tier

  • Up to 3 agents
  • Basic model access
  • Community support
  • 1,000 messages per month

Pro Plan

  • Unlimited agents
  • Advanced model access
  • Priority support
  • 50,000 messages per month
  • Custom plugins

Enterprise Plan

  • Everything in Pro
  • Dedicated infrastructure
  • SLA guarantees
  • Custom integrations
  • Team management

Configuring Your Profile

API Keys

Generate API keys from your dashboard:

  1. Navigate to Settings > API Keys
  2. Click Generate New Key
  3. Give your key a descriptive name
  4. Copy and securely store the key

Important: API keys are only shown once. Store them securely!

Team Settings

For team accounts:

  1. Go to Settings > Team
  2. Invite members via email
  3. Assign roles and permissions
  4. Set up billing preferences

Billing Information

Updating Payment Methods

  1. Navigate to Settings > Billing
  2. Click Payment Methods
  3. Add or update credit card information
  4. Set as default if needed

Viewing Invoices

All invoices are available in Settings > Billing > Invoice History where you can:

  • View invoice details
  • Download PDF copies
  • Update payment methods
  • Set up automatic payments
